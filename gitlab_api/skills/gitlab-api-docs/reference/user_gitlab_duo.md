# GitLab Duo
GitLab Duo is a suite of AI-native features that assist you while you work in GitLab. These features aim to help increase velocity and solve key pain points across the software development lifecycle.
GitLab Duo features are available in [IDE extensions](https://docs.gitlab.com/editor_extensions/) and the GitLab UI. Some features are also available as part of [GitLab Duo Chat](https://docs.gitlab.com/user/gitlab_duo_chat/examples/).
What do you want to do?
Search multiple sources, create files, and take action.
Use pre-built or custom agents to handle specialized work.
Chain together agents to solve complex problems automatically.
Create structured representations of your repositories to improve AI accuracy.
Integrate GitLab Duo with third-party data sources.
Customize GitLab Duo to match your workflow, standards, or requirements.
Host your own AI gateway and language models.
Ensure GitLab Duo is configured and operating properly.
